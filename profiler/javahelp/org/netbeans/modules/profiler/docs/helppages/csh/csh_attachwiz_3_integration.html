Attach Wizard: Choose Integration Type

See Also

The Attach Wizard helps you configure the settings for profiling an application using the attach method. The attach settings are asociated with the project.

You use this page of the Attach Wizard to specify the method for integrating with the local server. You can choose how to integrate with the server if the target server is the Sun Java System Application Server/GlassFish, the Apache Tomcat web server or the JBoss Application Server.

In this page of the wizard you specify one of the following integration types:

  • Automatic. If you choose this integration method, the IDE performs the necessary configuration to enable the IDE to attach to the specified server. You will need to provide the installation location of the server in the next step.
  • Manual. If you choose this integration method, you will need to follow the instructions provided in the next step to manually configure the server for attaching the IDE.
