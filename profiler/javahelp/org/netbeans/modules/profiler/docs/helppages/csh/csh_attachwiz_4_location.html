Attach Wizard: Locate Required Files

See Also

The Attach Wizard helps you configure the settings for profiling an application using the attach method. The attach settings are asociated with the project.

You use this page of the Attach wizard to specify the server installation location and JVM platform when using the automatic integration method.

In this page of the wizard you specify the following server settings:

  • Server Installation. Type the location of the installation directory of the target server or use the browse button to locate the directory on your machine.
  • Domain. Select the domain from the drop-down list. The drop-down list is populated automatically based on the specified server installation directory.

In this page of the wizard you also specify the following JVM platform setting:

  • JVM Platform. Select the JVM platform from the list of platforms registered with the IDE. You can add an additional platform by clicking Manage Platforms and adding a platform in the Platform Manager.
