Attach Wizard: Review Integration

See Also

The Attach Wizard helps you configure the settings for profiling an application using the attach method. The attach settings are asociated with the project.

You use this page of the Attach wizard to review the changes the IDE makes to the server configuration to enable attaching to the server. You see this page when you use the automatic integration method. The information pane describes the changes the IDE will make to configure the target server when you click Perform.

To perform the server integration:

  1. Review the changes that the IDE will make to the server configuration.
  2. Click Perform.

When you click Perform, the IDE makes the described changes to the configuration of the target server. After the IDE performs the changes, click Next to review the steps you need to perform to start the server in attach mode and profile the application.

  • When you want to revert the changes made to the server configuration, you need to revert the changes manually. This pane describes how to restore the original configuration. You may want to copy the instructions to reference later.
