Attach Wizard: Review Additional Steps

See Also

The Attach Wizard helps you configure the settings for profiling an application using the attach method. The attach settings are asociated with the project. You see this page when you use the automatic integration method.

You use this page of the Attach wizard to review the steps necessary to start the server in attach mode and start the profiling session.

In this page you should review the steps for starting the server and profiling session and do one of the following:

  • Click Finish and start the server with the command option given in the first step.
  • Select "Automatically start the server" and click Finish. In this case the IDE starts the server for you after you select the profiling task.

When you click Finish, the Attach wizard closes. To start the profiling session, choose the profiling task in the Select Profiling Task dialog box and click Attach. When you click Attach, the server is started in attach mode and the profiling session starts.
