Select Profiling Task Dialog Box

See Also

The Select Profiling Task dialog box is the main interface for selecting and running profiling tasks. You cannot run more than one profiling session at one time.

You open the dialog box by choosing Profile > Profile Main Project in the main menu.

The Select Profiling Task dialog box enables you to choose the following profiling tasks.

  • Monitor Application. This profile task returns high-level information about several important properties of the target JVM, including thread activity and memory allocations.
  • Analyze CPU Performance. This profile task returns detailed data on application performance. You can choose to analyze the entire application, part of the application, or startup performance.
  • Analyze Memory Usage. This profile task returns data on object allocation and garbage collection.

When you select a profiling task you can then set properties for that task in the right pane of the dialog box.

You can also create a customized configuration based on one of the profiling task by selecting a task and then clicking Create Custom. A custom configurations enables you to have greater control over the internal settings for that profiling task.

When you click Run, the target application launches and the selected profiling task starts.
