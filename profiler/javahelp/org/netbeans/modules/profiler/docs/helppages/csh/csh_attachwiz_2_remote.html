Attach Wizard: Remote System

See Also

The Attach Wizard helps you configure the settings for profiling an application using the attach method. The attach settings are asociated with the project.

You use this page of the Attach wizard to specify the details of the remote system. The remote system is the remote machine that is the deployment target of your application.

You see this page if you chose Remote as the attach method in the previous page of the wizard. If the target machine is your local machine, click Back and change the attach method to Local.

To profile an application on a remote machine you need to configure the JVM of the remote machine in order to attach the IDE.

In this page of the wizard you specify the following settings:

  • Hostname. Type the location of the target machine.
  • Host OS. Select the operating system of the target machine.

Notes:

  • The Dynamic attach method requires that the target application is running on JDK 1.6.
  • The Dynamic attach method is not available if you are profiling a remote application.
